1. What is the difference between var, let, and const?
Ans: var is function-scoped and can be redeclared, let is block-scoped and can be updated but not redeclared, and const is block-scoped and cannot be reassigned.

2. What is the spread operator (...)?
Ans: The spread operator (...) is used to expand elements of an array or properties of an object into another array, object, or function arguments.

3. What is the difference between map(), filter(), and forEach()?
Ans: map() creates a new array by transforming elements, filter() creates a new array with elements that match a condition, and forEach() runs a function for each element without returning a new array.

4. What is an arrow function?
Ans: An arrow function is a shorter syntax for writing functions in JavaScript using =>.

5. What are template literals?
Ans: Template literals are strings written with backticks (`) that allow variable interpolation using ${} and multi-line strings.
